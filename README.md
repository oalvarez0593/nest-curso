# nest-curso
Curso nest de la plataforma Udemy con el profesor Fernando Herrera.

#Ejecutar en desarrollo
1. Clonar el repositorio.
2. Ejecutar
```
npm install
```
3. Tener nest CLI instalado
```
npm i -g @nestjs/cli
```

4. Levantar la base de datos
```
docker-compose up -d
```


## Stack usado 
* MongoDB
* NEST
